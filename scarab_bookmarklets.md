# SCARAB Bookmarklets — Works on ANY Browser (No Extensions Needed)

Bookmarklets are simple bookmarks that run JavaScript when clicked.
They work on every browser — Chrome, Safari, Firefox, Lemur, Samsung Internet, etc.
No Tampermonkey, no extensions needed.

## How to install:

1. Copy the entire "javascript:..." code below
2. Create a new bookmark in your browser
3. Set the URL/paste the code as the bookmark URL
4. Click the bookmark when you want to use it

## Instructions:

1. **Bookmarklet method** (recommended for mobile):
   - Copy one of the javascript: blocks above
   - Create a new bookmark in your browser
   - Set the name to "SCARAB Quora" or "SCARAB X"
   - Paste the javascript: code as the URL
   - Click the bookmark when you're on the site

2. **Tampermonkey method** (for desktop Chrome/Firefox):
   - Install Tampermonkey extension
   - Click Tampermonkey → Create new script
   - Delete the default code, paste the content of any `.user.js` file
   - File → Save

3. **For Lemur Browser specifically**:
   - Try bookmarklet method first (no extension needed)
   - If you want Tampermonkey, try `scarab_grant_none.user.js` (v5) which uses @grant none
   - Make sure the script is enabled in Tampermonkey dashboard
