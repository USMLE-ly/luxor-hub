====================================================================
 DOCKER RUN — WITH CAPABILITIES FOR CAMOUFOX / DONUT BROWSER
====================================================================

Run this on the HOST machine to recreate the container with 
proper capabilities for browser automation:

  docker run -it --rm \
    --cap-add=SYS_PTRACE \
    --cap-add=SYS_ADMIN \
    --cap-add=NET_ADMIN \
    --cap-add=SYS_NICE \
    --security-opt seccomp=unconfined \
    --security-opt apparmor=unconfined \
    -v /root/Documents/Codex/2026-06-08/make-the-deepseek-v4-flash-free:/workspace \
    -e DISPLAY=:99 \
    your-image-name

Once inside the container, run:

  cd /workspace
  source .venv/bin/activate
  bash launch_donut_quora.sh
